# ⚡ Reaction Dash

ランダムに光る数字キー（0〜9）を素早くタイプして敵を倒す、キーボード反応速度バトルゲーム。

## ✨ 特徴

- **キーボード操作**: 0〜9の数字キーでプレイ
- **敵キャラクター**: ラウンドごとに異なる敵とバトル
- **HPシステム**: 正解キーを押すと敵にダメージ
- **難易度選択**: Calm / Normal / Intense の3段階
- **コンボシステム**: 連続正解でコンボ倍率アップ
- **ラウンド制**: 5ラウンドをクリアして全制覇
- **スコア表示**: 平均反応速度、最速、命中率を記録
- **達成演出**: ラウンドクリア時のポップアップとスコア表示

## 🎮 遊び方

1. **セッション開始**ボタンを押す
2. ランダムに光る数字キー（0〜9）を確認
3. 光った数字と同じキーを素早くタイプ
4. 敵のHPを0にしてラウンドクリア
5. 全5ラウンドをクリアしてゲームクリア

### 操作方法

- **数字キー（0〜9）**: 光った数字と同じキーをタイプ
- **セッション開始**: ゲーム開始
- **一時停止**: ゲームを一時停止
- **リセット**: スコアとラウンドをリセット
- **テンポ選択**: 難易度を変更（Calm / Normal / Intense）

### 難易度

- **Calm**: 1.2〜2.0秒間隔（ゆっくり）
- **Normal**: 0.85〜1.5秒間隔（標準）
- **Intense**: 0.52〜1.1秒間隔（激速）

## 🎯 ラウンド構成

- **Round 1**: 敵HP 10
- **Round 2**: 敵HP 12
- **Round 3**: 敵HP 14
- **Round 4**: 敵HP 16
- **Round 5**: 敵HP 18

各ラウンドで敵のHPが増加し、難易度が上がります。

## 📊 スコア項目

- **平均反応速度**: 全ヒットの平均反応時間（ms）
- **最速反応**: 最も速かった反応時間（ms）
- **命中率**: 正解率（%）
- **最大コンボ**: 連続正解の最大数

## 🛠️ 技術スタック

- **HTML5**: セマンティックマークアップ
